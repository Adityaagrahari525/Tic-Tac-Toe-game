🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.
This project allows two players to play the classic Tic Tac Toe game in the browser.

📌 Features

Two player game (Player X and Player O)

Interactive game board

Automatic winner detection

Displays winner message

Reset game functionality

New game option

Prevents clicking the same box twice

🛠️ Technologies Used

HTML – Structure of the game

CSS – Styling of the board and buttons

JavaScript – Game logic and winner detection

🎯 How to Play

The game has a 3 × 3 grid.

Player O starts the game.

Players take turns clicking on empty boxes.

The first player to get 3 marks in a row wins.

Horizontally

Vertically

Diagonally

If all boxes are filled and no player wins, the game ends in a draw.
